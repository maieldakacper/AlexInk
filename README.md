# Projekt do portfolio nr. 2 #
# ALEX INK - Tattoo Portfolio #

Wielostronicowa (Multi-page) wizytówka w stylu "Dark Mode", zaprojektowana dla niezależnego tatuatora. 

Projekt wykorzystuje czysty HTML i CSS. Największymi atutami strony są: w pełni responsywna, asymetryczna galeria zdjęć (stworzona z użyciem CSS `column-count`, bez ciężkich skryptów JS) oraz interaktywne FAQ (zbudowane na natywnych tagach HTML). Strona stawia na UX dopasowany do branży, kierując klientów bezpośrednio na social media artysty.

👉 **Zobacz na żywo:** [https://maieldakacper.github.io/AlexInk/]




# Portfolio Project No. 2 #
# ALEX INK - Tattoo Portfolio #

A multi-page "Dark Mode" showcase website designed for an independent tattoo artist.

The project utilizes clean HTML and CSS. The site's key strengths include a fully responsive, asymmetrical photo gallery (created using CSS column-count without heavy JS scripts) and an interactive FAQ section built with native HTML tags. The design prioritizes industry-specific UX, effectively directing clients to the artist's social media profiles.

👉 View Live: [https://maieldakacper.github.io/AlexInk/]
